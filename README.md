# News App

This is a simple news application that allows users to browse and read the latest news articles. The application fetches news data from the [News API](https://newsapi.org/) and displays it in a user-friendly format.

## Features

- Browse top headlines from various news sources.
- Search for news articles using keywords.
- Read full news articles by clicking on them.
- Responsive design for mobile and desktop.

## Technologies Used

- HTML
- CSS
- JavaScript
- News API
